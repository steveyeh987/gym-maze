# Installation

1. Install [OpenAI Gym](https://github.com/openai/gym) and its dependencies.
2. Install the package itself:
```
git clone https://github.com/changlintsai/gym-maze.git
cd gym-maze
pip install -e .
```

# Use your environment

```
import gym
import gym_maze
env = gym.make("maze-v0")
```
